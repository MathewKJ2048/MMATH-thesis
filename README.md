# MMATH-thesis
LaTeX source files for MMATH Thesis at UWaterloo

# Build instructions:

Run the following:

```
git clone https://github.com/MathewKJ2048/MMATH-thesis
cd ./MMATH-thesis/
make
```

The thesis is generated as `uw-ethesis.pdf` in the root folder.
